# HotelXpress_UI

This is simple interface foe HotelXPress 

# Getting started

- Install all dependeces with 
```sh
npm install  
```

- That was all you need to do, now run the project with :

```sh
npm start 
```

### Two User that you can use ( you can create your own user )
